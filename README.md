# \# DentalLab API

# 

# A RESTful API for dental laboratory management built with ASP.NET Core 8.

# 

# \## Technologies

# \- ASP.NET Core 8

# \- Entity Framework Core 8

# \- SQLite

# \- Swagger / OpenAPI

# 

# \## Endpoints

# 

# \### Patients

# \- GET /api/patients

# \- GET /api/patients/{id}

# \- POST /api/patients

# \- PUT /api/patients/{id}

# \- DELETE /api/patients/{id}

# 

# \### Orders

# \- GET /api/orders

# \- GET /api/orders/{id}

# \- GET /api/orders/status/{status}

# \- POST /api/orders

# \- PUT /api/orders/{id}

# \- PUT /api/orders/{id}/status

# \- DELETE /api/orders/{id}

# 

# \## Run Locally

# dotnet run

# Navigate to http://localhost:5221/swagger
